#### Charla: Code peer review en la práctica: cómo revisamos código en proyectos abiertos
📅 **Fecha**: 26 de Marzo 2026

**Oradora:** Yanina Bellini Saibene Community, Manager de rOpenSci, Project Lead de R-Ladies y profesora de la Universidad Austral.

**Breve descripción:**
En esta charla Yanina compartió cómo realizan code peer review en rOpenSci, combinando la revisión humana con herramientas automáticas. A partir de ejemplos reales, presento el uso de pkgcheck para automatizar los chequeos de calidad y enfocar la revisión en decisiones que realmente importan. La charla fue práctica y pensada para que RSE y equipos de RSE se lleven ideas concretas y aplicables a sus propios flujos de trabajo.
